DotDot is a CSS pre-processor.

# Tutorials

- [How to create a CSS pre-processor from scratch (DotDot)](https://dev.to/abdurrahmaanj/how-to-build-a-css-pre-processor-like-sass-from-scratch-33mc)

# Syntax

## Variables

- Normal
```
x = 1
```

- Supports -
```
x-y-z = 1
```

## Functions

- supports statements

```
@f{
    border-radius: 50%;
}
```

- supports variables

```
@f{
    border-radius: 50%;
    border: ..size solid black;
}
```
